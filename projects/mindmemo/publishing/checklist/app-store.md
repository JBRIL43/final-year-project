# App Store Submission Checklist

- [ ] Apple Developer account active
- [ ] Bundle ID and signing configured
- [ ] Privacy Nutrition Labels completed (no data linked to user off-device)
- [ ] Microphone usage description in Info.plist
- [ ] On-device AI / model download disclosed in review notes if applicable
- [ ] TestFlight build tested on physical device
- [ ] Screenshots per `../store-assets/screenshots-brief.md`
- [ ] Listing copy per `../store-assets/listing.md`
- [ ] Export compliance answered (likely no encryption beyond HTTPS for store tools only)
- [ ] Final smoke: record → transcribe → tag → analyze → chat → convert
